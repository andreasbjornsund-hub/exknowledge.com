كيفية قراءة العلامات الموجودة على المعدات

يحمل كل جهاز Ex معتمد علامة تشفر نوع الحماية ومجموعة الغاز/الغبار وفئة درجة الحرارة ومستوى حماية المعدات. يعد فهم هذه العلامات أمرًا ضروريًا لاختيار المعدات وتركيبها وفحصها.

هيكل علامة ATEX

تحمل معدات ATEX علامة منظمة وفقًا للتوجيه 2014/34/EU:


CE 0539 ⟨Ex⟩ II 2G Ex db IIC T4 Gb
│ │ │ │ │ │ │ │ │ │
│ │ │ │ │ │ │ │ │ └─ EPL (جيجابايت = عالي، المنطقة 1)
│ │ │ │ │ │ │ │ └──── فئة درجة الحرارة (T4 = 135 درجة مئوية كحد أقصى)
│ │ │ │ │ │ │ └──────── مجموعة الغاز (IIC = الهيدروجين/الأسيتيلين)
│ │ │ │ │ │ └───────── نوع الحماية (db = مقاوم للاشتعال، EPL b)
│ │ │ │ │ └──────────── رمز Ex (محمي من الانفجار)
│ │ │ │ └───────────────── الفئة (2G = الفئة 2، غاز)
│ │ │ └──────────────────── مجموعة المعدات (II = الصناعات السطحية)
│ │ └────────────────────────── رمز "Ex" السداسي ATEX
│ └───────────────────────────── رقم الهيئة المبلغ عنها
└───────────────────────────────── علامة CE

هيكل وضع العلامات IECEx

تتبع علامات IECEx المواصفة IEC 60079-0:


السابق ديسيبل IIC T4 جيجابايت
│ │ │ │ │
│ │ │ │ └─ الدوري الإنجليزي
│ │ │ └──── فئة درجة الحرارة
│ │ └──────── مجموعة الغاز
│ └─────────── نوع الحماية مع لاحقة المستوى
└────────────── الرمز السابق

لا تتضمن علامات IECEx علامة CE أو رقم المجموعة أو الفئة. بدلاً من ذلك، يوفر رقم شهادة IECEx إمكانية التتبع:


IECEx BAS 12.0034X
│ │ │
│ │ └─ رقم الشهادة + المراجعة (X = شروط محددة)
│ └───── كود ExCB (BAS = Baseefa, UK)
└─────────── معرف مخطط IECEx

قراءة نوع الحماية

يستخدم نوع الحماية البادئة "Ex" المكونة من حرفين متبوعة بحرف الطريقة ولاحقة المستوى:

بمناسبة الطريقة مستوى معنى
على سبيل المثال السلامة الجوهرية a المنطقة 0 قادرة
السابقين السلامة الجوهرية b المنطقة 1 قادرة
على سبيل المثال السلامة الجوهرية c المنطقة 2 فقط
السابق ديسيبل مقاوم للاشتعال b المنطقة 1 قادرة
العاصمة السابقة مقاوم للاشتعال c المنطقة 2 فقط
السابق إب زيادة الأمان b المنطقة 1 قادرة
على سبيل المثال زيادة الأمان c المنطقة 2 فقط
السابق أماه التغليف a المنطقة 0 قادرة
السابق ميغابايت التغليف b المنطقة 1 قادرة
السابق مولودية التغليف c المنطقة 2 فقط
السابق بكسل مضغوط x المنطقة 1 → غير خطرة
السابق الحمر مضغوط y المنطقة 1 → المنطقة 2
السابقين pz مضغوط z المنطقة 2 → غير هاز
على سبيل المثال لا غير اثارة — المنطقة 2 فقط
السابق nC اتصالات مغلقة / مختومة — المنطقة 2 فقط
السابقين nR التنفس المحدود — المنطقة 2 فقط
السابق تا حاوية غبار a المنطقة 20 قادرة
السابق السل حاوية غبار b المنطقة 21 قادرة
السابق ح حاوية غبار c المنطقة 22 فقط
السابق س غمر الزيت — المنطقة 1 قادرة
مثال ف تعبئة البودرة — المنطقة 1 قادرة

العلامات المجمعة

المعدات ذات طرق الحماية المتعددة:


السابق ديسيبل إب IIC T4 جيجابايت

الجسم الرئيسي: مقاوم للاشتعال (ديسيبل). حجرة المحطة الطرفية: زيادة السلامة (eb).


Ex db [ib Ga] IIC T4 Gb

حاوية مقاومة للاشتعال (Gb) تحتوي على دوائر آمنة جوهريًا (Ga). تشير الأقواس إلى دوائر IS بالداخل.


السابق ديسيبل IIC T6...T4 جيجا بايت

درجة الحرارة تختلف مع درجة الحرارة المحيطة. T6 في البيئة المحيطة القياسية، T4 في البيئة المحيطة المرتفعة.

علامات مزدوجة للغاز والغبار

المعدات المعتمدة لكل من الغاز والغبار:


II 2G Ex ديسيبل IIC T4 جيجابايت
II 2D Ex tb IIIC T125°C ديسيبل

يظهر كلا الخطين على لوحة الاسم. يمكن استخدام المعدات في منطقة الغاز 1 أو منطقة الغبار 21.

قراءة أمثلة من العالم الحقيقي

مثال 1: صندوق التوصيل


CE 0539 ⟨Ex⟩ II 2G Ex eb IIC T5 Gb
  • ATEX المجموعة الثانية، الفئة 2G (غاز المنطقة 1)
  • زيادة الأمان (Ex eb)
  • جميع المجموعات الغازية (IIC)
  • T5 (أقصى سطح 100 درجة مئوية)
  • الدوري الانجليزي جي بي

مثال 2: المصباح الكاشف LED


CE 0539 ⟨Ex⟩ II 2G Ex db eb IIC T4 Gb
                    II 2D Ex tb IIIC T135°C ديسيبل
  • التصنيف المزدوج: الغاز (المنطقة 1) والغبار (المنطقة 21)
  • جسم مقاوم للاشتعال مع صندوق طرفي آمن متزايد
  • جميع مجموعات الغاز (IIC)، جميع مجموعات الغبار (IIIC)
  • الغاز: T4 (135 درجة مئوية كحد أقصى)، الغبار: 135 درجة مئوية كحد أقصى

مثال 3: حافظة iPhone معتمدة من ATEX (Xshielder)


⟨Ex⟩ II 2G Ex db IIC T4 Gb
     II 2D Ex tb IIIC T135°C ديسيبل
  • المنطقة 1 غاز + المنطقة 21 غبار
  • غلاف مقاوم للاشتعال + محمي من الغبار
  • يغطي جميع مجموعات الغاز ومجموعات الغبار
  • تصنيف IP65، متوافق مع الشحن اللاسلكي

مثال 4: جهاز إرسال آمن جوهريًا


CE 0539 ⟨Ex⟩ II 1G Ex ia IIC T4 Ga
  • الفئة 1G — مناسبة للمنطقة 0
  • آمن جوهريًا (أعلى مستوى، على سبيل المثال)
  • جميع المجموعات الغازية (IIC)
  • T4 (135 درجة مئوية كحد أقصى)
  • EPL Ga ( حماية عالية جدا )

مثال 5: محرك المنطقة 2


CE 0539 ⟨Ex⟩ II 3G Ex nA IIB T3 Gc
  • الفئة 3G - المنطقة 2 فقط
  • عدم اثارة (على سبيل المثال لا يوجد)
  • مجموعات الغاز IIA وIIB فقط (وليس IIC)
  • T3 (200 درجة مئوية كحد أقصى)
  • الدوري الإنجليزي الممتاز

علامات إضافية على لوحة الاسم

بالإضافة إلى علامة Ex، تتضمن ملصقات المعدات ما يلي:

بمناسبة معنى
تا أو تامب نطاق درجة الحرارة المحيطة (على سبيل المثال، -40 درجة مئوية إلى +60 درجة مئوية)
IP66 / IP68 تصنيف حماية الدخول
X شروط الاستخدام الخاصة (انظر الشهادة)
U شهادة المكون (غير مستقلة)
رقم الشهادة إشارة إلى شهادة الاختبار الكاملة
الرقم التسلسلي معرف الجهاز الفريد للتتبع
سنة الصنع مطلوب لمسار التدقيق

ال "X" اللاحقة مهمة – فهي تعني أن هناك شروطًا خاصة موثقة في الشهادة يجب مراعاتها أثناء التثبيت والاستخدام. قم دائمًا باسترداد الشهادة وقراءتها عندما ترى "X".

علامات NEC/CEC (أمريكا الشمالية)

تتبع علامة أمريكا الشمالية تنسيقًا مختلفًا:


الدرجة الأولى، القسم 1، المجموعات ب، ج، د، T4

أو في نظام المنطقة :


الفئة الأولى، المنطقة 1، AEx db IIC T4 Gb

تشير البادئة "A" قبل "Ex" إلى أن الجهاز معتمد وفقًا لمعايير أمريكا الشمالية (NEC/CEC) باستخدام نظام المنطقة.

ملفات ذات صلة

  • طرق الحماية - شرح تفصيلي لكل طريقة
  • مجموعات الغاز — ماذا يعني IIA/IIB/IIC
  • درجات الحرارة - ماذا يعني T1 – T6
  • الدوري الانجليزي - ماذا يعني Ga/Gb/Gc
  • المعايير - ما هو المعيار الذي يحكم كل عنصر تمييز
  • شهادة - كيفية إصدار الشهادات والعلامات
