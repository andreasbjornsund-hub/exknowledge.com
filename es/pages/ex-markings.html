Cómo leer el marcado Ex en equipos

Cada equipo Ex certificado tiene un marcado que codifica el modo de protección, grupo de gas/polvo, clase de temperatura y nivel de protección del equipo. Comprender estas marcas es esencial para seleccionar, instalar e inspeccionar equipos.

Estructura del marcado ATEX

Los equipos ATEX tienen un marcado estructurado según la Directiva 2014/34/UE:


CE 0539  ⟨Ex⟩  II 2G  Ex db IIC T4 Gb
│   │     │     │  │    │  │  │   │  │
│   │     │     │  │    │  │  │   │  └─ EPL (Gb = Alto, Zona 1)
│   │     │     │  │    │  │  │   └──── Clase de temperatura (T4 = máx. 135 °C)
│   │     │     │  │    │  │  └──────── Grupo de gas (IIC = hidrógeno/acetileno)
│   │     │     │  │    │  └─────────── Modo de protección (db = antideflagrante, EPL b)
│   │     │     │  │    └────────────── Símbolo Ex (protección contra explosiones)
│   │     │     │  └─────────────────── Categoría (2G = Categoría 2, Gas)
│   │     │     └────────────────────── Grupo de equipo (II = industria de superficie)
│   │     └──────────────────────────── Símbolo ATEX hexagonal «Ex»
│   └────────────────────────────────── Número del Organismo Notificado
└────────────────────────────────────── Marcado CE

Estructura del marcado IECEx

Los marcados IECEx siguen IEC 60079-0:

ExdbIICT4Gb
↓↓↓↓↓
Ex symbolProtection typeGas groupTemp classEPL

Los marcados IECEx no incluyen el marcado CE, número de grupo ni categoría. En su lugar, el número de certificado IECEx proporciona la trazabilidad:

IECExBAS12.0034X
↓↓↓
IECEx schemeExCB code
(Baseefa, UK)		Certificate + revision
(X = specific conditions)

Leer el modo de protección

El modo de protección utiliza el prefijo de dos letras "Ex" seguido de la letra del método y el sufijo de nivel:

Marcados combinados

Equipos con múltiples métodos de protección:


Ex db eb IIC T4 Gb

Parte principal: antideflagrante (db). Compartimento de terminales: seguridad aumentada (eb).


Ex db [ib Ga] IIC T4 Gb

Envolvente antideflagrante (Gb) que contiene circuitos intrínsecamente seguros (Ga). Los corchetes indican los circuitos IS en el interior.


Ex db IIC T6...T4 Gb

La clase de temperatura varía con la temperatura ambiente. T6 a temperatura ambiente estándar, T4 a temperatura ambiente elevada.

Marcado doble de gas y polvo

Equipos certificados tanto para gas como para polvo:


II 2G  Ex db IIC T4 Gb
II 2D  Ex tb IIIC T125°C Db

Ambas líneas aparecen en la placa de características. El equipo puede usarse en zona de gas 1 o zona de polvo 21.

Lectura de ejemplos reales

Ejemplo 1: Caja de conexiones


CE 0539 ⟨Ex⟩ II 2G Ex eb IIC T5 Gb
  • ATEX Grupo II, Categoría 2G (Zona 1 gas)
  • Seguridad aumentada (Ex eb)
  • Todos los grupos de gas (IIC)
  • T5 (máx. 100 °C de superficie)
  • EPL Gb

Ejemplo 2: Proyector LED


CE 0539 ⟨Ex⟩ II 2G Ex db eb IIC T4 Gb
                    II 2D Ex tb IIIC T135°C Db
  • Doble aprobación: gas (Zona 1) y polvo (Zona 21)
  • Cuerpo antideflagrante con caja de terminales de seguridad aumentada
  • Todos los grupos de gas (IIC), todos los grupos de polvo (IIIC)
  • Gas: T4 (135 °C máx.), Polvo: 135 °C máx.

Ejemplo 3: Funda de iPhone certificada ATEX (Xshielder)


⟨Ex⟩ II 2G Ex db IIC T4 Gb
     II 2D Ex tb IIIC T135°C Db
  • Zona 1 gas + Zona 21 polvo
  • Envolvente antideflagrante + protección antipolvo
  • Cubre todos los grupos de gas y polvo
  • Clasificación IP65, compatible con carga inalámbrica

Ejemplo 4: Transmisor intrínsecamente seguro


CE 0539 ⟨Ex⟩ II 1G Ex ia IIC T4 Ga
  • Categoría 1G — apto para Zona 0
  • Intrínsecamente seguro (nivel más alto, Ex ia)
  • Todos los grupos de gas (IIC)
  • T4 (135 °C máx.)
  • EPL Ga (protección muy alta)

Ejemplo 5: Motor de Zona 2


CE 0539 ⟨Ex⟩ II 3G Ex nA IIB T3 Gc
  • Categoría 3G — solo Zona 2
  • No generador de chispas (Ex nA)
  • Grupos de gas IIA e IIB solamente (no IIC)
  • T3 (200 °C máx.)
  • EPL Gc

Marcados adicionales en la placa de características

Además del marcado Ex, las etiquetas del equipo incluyen:

Marcado Significado
Ta o TambRango de temperatura ambiente (ej. −40 °C a +60 °C)
IP66 / IP68Grado de protección del envolvente
XCondiciones especiales de uso (ver certificado)
UCertificado de componente (no es independiente)
N.º certificadoReferencia al certificado de ensayo completo
N.º de serieIdentificador único de unidad para trazabilidad
Año de fabricaciónRequerido para trazabilidad de revisiones

El sufijo "X" es importante — significa que existen condiciones especiales documentadas en el certificado que deben cumplirse durante la instalación y el uso. Siempre obtenga y lea el certificado cuando vea "X".

Marcados NEC/CEC (Norteamérica)

El marcado norteamericano sigue un formato diferente:


Class I, Division 1, Groups B, C, D, T4

O en el sistema de zonas:


Class I, Zone 1, AEx db IIC T4 Gb

El prefijo "A" antes de "Ex" indica que el equipo está certificado según normas norteamericanas (NEC/CEC) con el sistema de zonas.
