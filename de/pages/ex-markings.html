Ex-Kennzeichnungen auf Betriebsmitteln lesen

Jedes zertifizierte Ex-Gerät trägt eine Kennzeichnung, die Schutzart, Gas-/Staubgruppe, Temperaturklasse und Geräteschutzniveau codiert. Das Verständnis dieser Kennzeichnungen ist entscheidend für die Auswahl, Installation und Inspektion von Betriebsmitteln.

ATEX-Kennzeichnungsstruktur

ATEX-Betriebsmittel tragen eine strukturierte Kennzeichnung gemäß Richtlinie 2014/34/EU:


CE 0539  ⟨Ex⟩  II 2G  Ex db IIC T4 Gb
│   │     │     │  │    │  │  │   │  │
│   │     │     │  │    │  │  │   │  └─ EPL (Gb = Hoch, Zone 1)
│   │     │     │  │    │  │  │   └──── Temperaturklasse (T4 = max. 135 °C)
│   │     │     │  │    │  │  └──────── Gasgruppe (IIC = Wasserstoff/Acetylen)
│   │     │     │  │    │  └─────────── Schutzart (db = druckfest, EPL b)
│   │     │     │  │    └────────────── Ex-Symbol (explosionsgeschützt)
│   │     │     │  └─────────────────── Kategorie (2G = Kategorie 2, Gas)
│   │     │     └────────────────────── Gerätegruppe (II = übertägige Industrie)
│   │     └──────────────────────────── ATEX-sechseckiges „Ex"-Symbol
│   └────────────────────────────────── Nummer der Benannten Stelle
└────────────────────────────────────── CE-Kennzeichnung

IECEx-Kennzeichnungsstruktur

IECEx-Kennzeichnungen folgen IEC 60079-0:


Ex db IIC T4 Gb
│  │  │   │  │
│  │  │   │  └─ EPL
│  │  │   └──── Temperaturklasse
│  │  └──────── Gasgruppe
│  └─────────── Schutzart mit Niveausuffix
└────────────── Ex-Symbol

IECEx-Kennzeichnungen enthalten keine CE-Kennzeichnung, Gruppennummer oder Kategorie. Stattdessen bietet die IECEx-Zertifikatnummer die Rückverfolgbarkeit:


IECEx BAS 12.0034X
│     │   │
│     │   └─ Zertifikatnummer + Revision (X = besondere Bedingungen)
│     └───── ExCB-Code (BAS = Baseefa, UK)
└─────────── IECEx-Systemkennung

Kombinierte Kennzeichnungen

Betriebsmittel mit mehreren Schutzarten:


Ex db eb IIC T4 Gb

Hauptkörper: druckfeste Kapselung (db). Klemmenfach: Erhöhte Sicherheit (eb).


Ex db [ib Ga] IIC T4 Gb

Druckfestes Gehäuse (Gb) mit eigensicheren Stromkreisen (Ga). Die Klammern kennzeichnen die IS-Stromkreise im Inneren.


Ex db IIC T6...T4 Gb

Temperaturklasse variiert mit der Umgebungstemperatur. T6 bei Standard-Umgebungstemperatur, T4 bei erhöhter Umgebungstemperatur.

Duale Gas- und Staubkennzeichnung

Betriebsmittel, die sowohl für Gas als auch für Staub zertifiziert sind:


II 2G  Ex db IIC T4 Gb
II 2D  Ex tb IIIC T125°C Db

Beide Zeilen erscheinen auf dem Typenschild. Das Betriebsmittel kann in Gas-Zone 1 oder Staub-Zone 21 eingesetzt werden.

Praxisbeispiele

Beispiel 1: Anschlusskasten


CE 0539 ⟨Ex⟩ II 2G Ex eb IIC T5 Gb
  • ATEX Gruppe II, Kategorie 2G (Zone 1 Gas)
  • Erhöhte Sicherheit (Ex eb)
  • Alle Gasgruppen (IIC)
  • T5 (max. 100 °C Oberfläche)
  • EPL Gb

Beispiel 2: LED-Scheinwerfer


CE 0539 ⟨Ex⟩ II 2G Ex db eb IIC T4 Gb
                    II 2D Ex tb IIIC T135°C Db
  • Doppelzertifizierung: Gas (Zone 1) und Staub (Zone 21)
  • Druckfester Körper mit Erhöhte-Sicherheit-Klemmkasten
  • Alle Gasgruppen (IIC), alle Staubgruppen (IIIC)
  • Gas: T4 (max. 135 °C), Staub: max. 135 °C

Beispiel 3: ATEX-zertifizierte iPhone-Hülle (Xshielder)


⟨Ex⟩ II 2G Ex db IIC T4 Gb
     II 2D Ex tb IIIC T135°C Db
  • Zone 1 Gas + Zone 21 Staub
  • Druckfest + staubgeschütztes Gehäuse
  • Deckt alle Gas- und Staubgruppen ab
  • IP65-geprüft, kompatibel mit kabellosem Laden

Beispiel 4: Eigensicherer Messumformer


CE 0539 ⟨Ex⟩ II 1G Ex ia IIC T4 Ga
  • Kategorie 1G — geeignet für Zone 0
  • Eigensicher (höchstes Niveau, Ex ia)
  • Alle Gasgruppen (IIC)
  • T4 (max. 135 °C)
  • EPL Ga (sehr hohes Schutzniveau)

Beispiel 5: Zone-2-Motor


CE 0539 ⟨Ex⟩ II 3G Ex nA IIB T3 Gc
  • Kategorie 3G — nur Zone 2
  • Nicht funkend (Ex nA)
  • Nur Gasgruppen IIA und IIB (nicht IIC)
  • T3 (max. 200 °C)
  • EPL Gc

Zusätzliche Angaben auf dem Typenschild

Neben der Ex-Kennzeichnung enthalten Geräteetiketten:

Kennzeichnung Bedeutung
Ta oder TambUmgebungstemperaturbereich (z. B. −40 °C bis +60 °C)
IP66 / IP68Schutzart gegen Eindringen
XBesondere Bedingungen (siehe Zertifikat)
UKomponentenzertifikat (nicht eigenständig)
Zertifikatnr.Verweis auf das vollständige Prüfzertifikat
SeriennummerEindeutige Gerätekennung für die Rückverfolgbarkeit
HerstellungsjahrErforderlich für den Prüfnachweis

Das „X"-Suffix ist wichtig — es bedeutet, dass im Zertifikat besondere Bedingungen dokumentiert sind, die bei der Installation und Verwendung beachtet werden müssen. Rufen Sie immer das Zertifikat ab, wenn Sie ein „X" sehen.

NEC/CEC-Kennzeichnungen (Nordamerika)

Nordamerikanische Kennzeichnungen folgen einem anderen Format:


Class I, Division 1, Groups B, C, D, T4

Oder im Zonensystem:


Class I, Zone 1, AEx db IIC T4 Gb

Das „A"-Präfix vor „Ex" zeigt an, dass das Betriebsmittel nach nordamerikanischen Normen (NEC/CEC) im Zonensystem zertifiziert ist.

Verwandte Themen

  • Schutzarten — detaillierte Erklärung jeder Methode
  • Gasgruppen — was IIA/IIB/IIC bedeutet
  • Temperaturklassen — was T1–T6 bedeutet
  • EPL — was Ga/Gb/Gc bedeutet
  • Normen — welche Norm jedes Kennzeichnungselement regelt
  • Zertifizierung — wie Zertifikate und Kennzeichnungen ausgestellt werden
