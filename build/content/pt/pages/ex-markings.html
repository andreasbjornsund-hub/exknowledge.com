Como Ler Marcações Ex em Equipamentos

Cada dispositivo Ex certificado possui uma marcação que codifica o tipo de proteção, grupo de gás/poeira, classe de temperatura e nível de proteção do equipamento. Compreender estas marcações é essencial para selecionar, instalar e inspecionar equipamentos.

Estrutura da Marcação ATEX

O equipamento ATEX possui uma marcação estruturada conforme a Diretiva 2014/34/UE:


CE 0539  ⟨Ex⟩  II 2G  Ex db IIC T4 Gb
│   │     │     │  │    │  │  │   │  │
│   │     │     │  │    │  │  │   │  └─ EPL (Gb = Elevado, Zona 1)
│   │     │     │  │    │  │  │   └──── Classe de temperatura (T4 = máx. 135°C)
│   │     │     │  │    │  │  └──────── Grupo de gás (IIC = hidrogénio/acetileno)
│   │     │     │  │    │  └─────────── Tipo de proteção (db = à prova de explosão, EPL b)
│   │     │     │  │    └────────────── Símbolo Ex (proteção contra explosões)
│   │     │     │  └─────────────────── Categoria (2G = Categoria 2, Gás)
│   │     │     └────────────────────── Grupo de Equipamento (II = indústrias de superfície)
│   │     └──────────────────────────── Símbolo hexagonal ATEX "Ex"
│   └────────────────────────────────── Número do Organismo Notificado
└────────────────────────────────────── Marcação CE

Estrutura da Marcação IECEx

As marcações IECEx seguem a IEC 60079-0:


Ex db IIC T4 Gb
│  │  │   │  │
│  │  │   │  └─ EPL
│  │  │   └──── Classe de temperatura
│  │  └──────── Grupo de gás
│  └─────────── Tipo de proteção com sufixo de nível
└────────────── Símbolo Ex

As marcações IECEx não incluem a marcação CE, número de grupo ou categoria. Em vez disso, o número do certificado IECEx fornece a rastreabilidade:


IECEx BAS 12.0034X
│     │   │
│     │   └─ Número do certificado + revisão (X = condições especiais)
│     └───── Código ExCB (BAS = Baseefa, UK)
└─────────── Identificador do esquema IECEx

Marcações Combinadas

Equipamento com múltiplos métodos de proteção:


Ex db eb IIC T4 Gb

Corpo principal: à prova de explosão (db). Compartimento de terminais: segurança aumentada (eb).


Ex db [ib Ga] IIC T4 Gb

Invólucro à prova de explosão (Gb) contendo circuitos intrinsecamente seguros (Ga). Os parênteses retos indicam os circuitos IS no interior.


Ex db IIC T6...T4 Gb

A classe de temperatura varia com a temperatura ambiente. T6 à temperatura ambiente padrão, T4 à temperatura ambiente elevada.

Marcação Dupla Gás e Poeira

Equipamento certificado para gás e poeira:


II 2G  Ex db IIC T4 Gb
II 2D  Ex tb IIIC T125°C Db

Ambas as linhas aparecem na placa de identificação. O equipamento pode ser utilizado na Zona 1 de gás ou na Zona 21 de poeira.

Leitura de Exemplos Reais

Exemplo 1: Caixa de Junção


CE 0539 ⟨Ex⟩ II 2G Ex eb IIC T5 Gb
  • ATEX Grupo II, Categoria 2G (Zona 1 gás)
  • Segurança aumentada (Ex eb)
  • Todos os grupos de gás (IIC)
  • T5 (máx. 100°C de superfície)
  • EPL Gb

Exemplo 2: Projetor LED


CE 0539 ⟨Ex⟩ II 2G Ex db eb IIC T4 Gb
                    II 2D Ex tb IIIC T135°C Db
  • Classificação dupla: gás (Zona 1) e poeira (Zona 21)
  • Corpo à prova de explosão com caixa de terminais de segurança aumentada
  • Todos os grupos de gás (IIC), todos os grupos de poeira (IIIC)
  • Gás: T4 (135°C máx.), Poeira: 135°C máx.

Exemplo 3: Capa de iPhone Certificada ATEX (Xshielder)


⟨Ex⟩ II 2G Ex db IIC T4 Gb
     II 2D Ex tb IIIC T135°C Db
  • Zona 1 gás + Zona 21 poeira
  • À prova de explosão + invólucro protegido contra poeira
  • Abrange todos os grupos de gás e de poeira
  • Classificação IP65, compatível com carregamento sem fios

Exemplo 4: Transmissor Intrinsecamente Seguro


CE 0539 ⟨Ex⟩ II 1G Ex ia IIC T4 Ga
  • Categoria 1G — adequado para Zona 0
  • Intrinsecamente seguro (nível mais elevado, Ex ia)
  • Todos os grupos de gás (IIC)
  • T4 (135°C máx.)
  • EPL Ga (proteção muito elevada)

Exemplo 5: Motor de Zona 2


CE 0539 ⟨Ex⟩ II 3G Ex nA IIB T3 Gc
  • Categoria 3G — apenas Zona 2
  • Sem faíscas (Ex nA)
  • Grupos de gás IIA e IIB apenas (não IIC)
  • T3 (200°C máx.)
  • EPL Gc

Marcações Adicionais na Placa de Identificação

Para além da marcação Ex, as etiquetas do equipamento incluem:

MarcaçãoSignificado
Ta ou TambGama de temperatura ambiente (p. ex., −40°C a +60°C)
IP66 / IP68Grau de proteção contra intrusão
XCondições especiais de utilização (ver certificado)
UCertificado de componente (não autónomo)
N.º certificadoReferência ao certificado de ensaio completo
Número de sérieIdentificador único do dispositivo para rastreabilidade
Ano de fabricoObrigatório para trilho de auditoria

O sufixo "X" é importante — significa que existem condições especiais documentadas no certificado que devem ser observadas durante a instalação e utilização. Obtenha e leia sempre o certificado quando vir "X".

Marcações NEC/CEC (América do Norte)

A marcação norte-americana segue um formato diferente:


Class I, Division 1, Groups B, C, D, T4

Ou no sistema de Zonas:


Class I, Zone 1, AEx db IIC T4 Gb

O prefixo "A" antes de "Ex" indica que o equipamento é certificado segundo as normas norte-americanas (NEC/CEC) utilizando o sistema de Zonas.
