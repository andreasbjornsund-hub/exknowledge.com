كيف تقرأ علامات Ex على المعدات

كل جهاز Ex معتمد يحمل علامة تُشفّر نوع حمايته ومجموعة الغاز/الغبار وفئة الحرارة ومستوى حماية المعدات. فهم هذه العلامات ضروري لاختيار المعدات وتركيبها وفحصها.

هيكل علامة ATEX

تحمل معدات ATEX علامة منظمة وفق التوجيه 2014/34/EU:


CE 0539  ⟨Ex⟩  II 2G  Ex db IIC T4 Gb
│   │     │     │  │    │  │  │   │  │
│   │     │     │  │    │  │  │   │  └─ EPL (Gb = عالٍ، المنطقة 1)
│   │     │     │  │    │  │  │   └──── فئة الحرارة (T4 = أقصى 135°C)
│   │     │     │  │    │  │  └──────── مجموعة الغاز (IIC = هيدروجين/أسيتيلين)
│   │     │     │  │    │  └─────────── نوع الحماية (db = مقاوم للهب، EPL b)
│   │     │     │  │    └────────────── رمز Ex (محمي من الانفجار)
│   │     │     │  └─────────────────── الفئة (2G = الفئة 2، غاز)
│   │     │     └────────────────────── مجموعة المعدات (II = صناعات سطحية)
│   │     └──────────────────────────── رمز ATEX السداسي "Ex"
│   └────────────────────────────────── رقم الجهة المُبلَّغة
└────────────────────────────────────── علامة CE

مجموعات معدات ATEX

المجموعة التطبيق
I التعدين (غاز المناجم)
II الصناعات السطحية — غاز/بخار
III الصناعات السطحية — غبار

فئات ATEX

العلامة الفئة EPL المناطق
1G الفئة 1 غاز Ga 0 و1 و2
2G الفئة 2 غاز Gb 1 و2
3G الفئة 3 غاز Gc 2
1D الفئة 1 غبار Da 20 و21 و22
2D الفئة 2 غبار Db 21 و22
3D الفئة 3 غبار Dc 22

هيكل علامة IECEx

تتبع علامات IECEx معيار IEC 60079-0:


Ex db IIC T4 Gb
│  │  │   │  │
│  │  │   │  └─ EPL
│  │  │   └──── فئة الحرارة
│  │  └──────── مجموعة الغاز
│  └─────────── نوع الحماية مع لاحقة المستوى
└────────────── رمز Ex

لا تتضمن علامات IECEx علامة CE أو رقم المجموعة أو الفئة. بدلاً من ذلك، يوفر رقم شهادة IECEx إمكانية التتبع:


IECEx BAS 12.0034X
│     │   │
│     │   └─ رقم الشهادة + المراجعة (X = شروط خاصة)
│     └───── رمز ExCB (BAS = Baseefa، المملكة المتحدة)
└─────────── معرف نظام IECEx

قراءة نوع الحماية

يستخدم نوع الحماية البادئة "Ex" متبوعة بحرف الطريقة ولاحقة المستوى:

العلامة الطريقة المستوى المعنى
Ex ia سلامة ذاتية a قادرة على المنطقة 0
Ex ib سلامة ذاتية b قادرة على المنطقة 1
Ex ic سلامة ذاتية c المنطقة 2 فقط
Ex db مقاوم للهب b قادر على المنطقة 1
Ex dc مقاوم للهب c المنطقة 2 فقط
Ex eb سلامة معززة b قادرة على المنطقة 1
Ex ec سلامة معززة c المنطقة 2 فقط
Ex ma تغليف a قادر على المنطقة 0
Ex mb تغليف b قادر على المنطقة 1
Ex mc تغليف c المنطقة 2 فقط
Ex px ضغط زائد x المنطقة 1 → غير خطرة
Ex py ضغط زائد y المنطقة 1 → المنطقة 2
Ex pz ضغط زائد z المنطقة 2 → غير خطرة
Ex nA غير منتج للشرر — المنطقة 2 فقط
Ex nC ملامسات مغلقة/محكمة — المنطقة 2 فقط
Ex nR تنفس محدود — المنطقة 2 فقط
Ex ta حاوية غبار a قادرة على المنطقة 20
Ex tb حاوية غبار b قادرة على المنطقة 21
Ex tc حاوية غبار c المنطقة 22 فقط
Ex o غمر في الزيت — قادر على المنطقة 1
Ex q حشو بالمسحوق — قادر على المنطقة 1

العلامات المشتركة

معدات بطرق حماية متعددة:


Ex db eb IIC T4 Gb

الجسم الرئيسي: مقاوم للهب (db). حجرة الأطراف: سلامة معززة (eb).


Ex db [ib Ga] IIC T4 Gb

حاوية مقاومة للهب (Gb) تحتوي على دوائر سلامة ذاتية (Ga). الأقواس تشير إلى دوائر IS الداخلية.


Ex db IIC T6...T4 Gb

فئة الحرارة تتغير مع درجة الحرارة المحيطة. T6 عند الحرارة المحيطة القياسية، T4 عند الحرارة المحيطة المرتفعة.

علامة مزدوجة للغاز والغبار

معدات معتمدة لكل من الغاز والغبار:


II 2G  Ex db IIC T4 Gb
II 2D  Ex tb IIIC T125°C Db

يظهر كلا السطرين على لوحة البيانات. يمكن استخدام المعدات في منطقة غاز 1 أو منطقة غبار 21.

قراءة أمثلة واقعية

مثال 1: صندوق توصيل


CE 0539 ⟨Ex⟩ II 2G Ex eb IIC T5 Gb
  • ATEX المجموعة II، الفئة 2G (المنطقة 1 غاز)
  • سلامة معززة (Ex eb)
  • جميع مجموعات الغاز (IIC)
  • T5 (أقصى حرارة سطح 100°C)
  • EPL Gb

مثال 2: كشاف LED


CE 0539 ⟨Ex⟩ II 2G Ex db eb IIC T4 Gb
                    II 2D Ex tb IIIC T135°C Db
  • تصنيف مزدوج: غاز (المنطقة 1) وغبار (المنطقة 21)
  • جسم مقاوم للهب مع صندوق أطراف بسلامة معززة
  • جميع مجموعات الغاز (IIC) وجميع مجموعات الغبار (IIIC)
  • غاز: T4 (أقصى 135°C)، غبار: أقصى 135°C

مثال 3: غلاف آيفون معتمد ATEX (Xshielder)


⟨Ex⟩ II 2G Ex db IIC T4 Gb
     II 2D Ex tb IIIC T135°C Db
  • المنطقة 1 غاز + المنطقة 21 غبار
  • مقاوم للهب + حاوية محمية من الغبار
  • يغطي جميع مجموعات الغاز والغبار
  • تصنيف IP65، متوافق مع الشحن اللاسلكي

مثال 4: مرسل بسلامة ذاتية


CE 0539 ⟨Ex⟩ II 1G Ex ia IIC T4 Ga
  • الفئة 1G — مناسب للمنطقة 0
  • سلامة ذاتية (أعلى مستوى، Ex ia)
  • جميع مجموعات الغاز (IIC)
  • T4 (أقصى 135°C)
  • EPL Ga (حماية عالية جداً)

مثال 5: محرك المنطقة 2


CE 0539 ⟨Ex⟩ II 3G Ex nA IIB T3 Gc
  • الفئة 3G — المنطقة 2 فقط
  • غير منتج للشرر (Ex nA)
  • مجموعات الغاز IIA وIIB فقط (ليس IIC)
  • T3 (أقصى 200°C)
  • EPL Gc

علامات إضافية على لوحة البيانات

بالإضافة إلى علامة Ex، تتضمن ملصقات المعدات:

العلامة المعنى
Ta أو Tamb نطاق درجة الحرارة المحيطة (مثل −40°C إلى +60°C)
IP66 / IP68 تصنيف حماية الدخول
X شروط استخدام خاصة (انظر الشهادة)
U شهادة مكون (ليست مستقلة)
رقم الشهادة مرجع لشهادة الاختبار الكاملة
الرقم التسلسلي معرف فريد للجهاز للتتبع
سنة التصنيع مطلوب لمسار التدقيق

لاحقة "X" مهمة — تعني وجود شروط خاصة موثقة في الشهادة يجب مراعاتها أثناء التركيب والاستخدام. احصل دائماً على الشهادة واقرأها عند رؤية "X".

علامات NEC/CEC (أمريكا الشمالية)

تتبع العلامات في أمريكا الشمالية تنسيقاً مختلفاً:


Class I, Division 1, Groups B, C, D, T4

أو في نظام المناطق:


Class I, Zone 1, AEx db IIC T4 Gb

البادئة "A" قبل "Ex" تشير إلى أن المعدات معتمدة وفق معايير أمريكا الشمالية (NEC/CEC) باستخدام نظام المناطق.

مواضيع ذات صلة

  • طرق الحماية — شرح تفصيلي لكل طريقة
  • مجموعات الغاز — ماذا يعني IIA/IIB/IIC
  • فئات الحرارة — ماذا يعني T1–T6
  • EPL — ماذا يعني Ga/Gb/Gc
  • المعايير — أي معيار يحكم كل عنصر في العلامة
  • الشهادات — كيف تُصدر الشهادات والعلامات
