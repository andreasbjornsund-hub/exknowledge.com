Come leggere le marcature Ex sulle apparecchiature

Ogni dispositivo Ex certificato porta una marcatura che codifica il tipo di protezione, il gruppo gas/polvere, la classe di temperatura e il livello di protezione dell'apparecchiatura. Comprendere queste marcature è essenziale per selezionare, installare e ispezionare le apparecchiature.

Struttura della marcatura ATEX

Le apparecchiature ATEX portano una marcatura strutturata secondo la Direttiva 2014/34/UE:


CE 0539  ⟨Ex⟩  II 2G  Ex db IIC T4 Gb
│   │     │     │  │    │  │  │   │  │
│   │     │     │  │    │  │  │   │  └─ EPL (Gb = Alto, Zona 1)
│   │     │     │  │    │  │  │   └──── Classe di temperatura (T4 = max 135°C)
│   │     │     │  │    │  │  └──────── Gruppo gas (IIC = idrogeno/acetilene)
│   │     │     │  │    │  └─────────── Tipo di protezione (db = a prova di esplosione, EPL b)
│   │     │     │  │    └────────────── Simbolo Ex (protezione contro le esplosioni)
│   │     │     │  └─────────────────── Categoria (2G = Categoria 2, Gas)
│   │     │     └────────────────────── Gruppo apparecchiatura (II = industrie di superficie)
│   │     └──────────────────────────── Simbolo esagonale ATEX "Ex"
│   └────────────────────────────────── Numero dell'Organismo notificato
└────────────────────────────────────── Marchio CE

Struttura della marcatura IECEx

Le marcature IECEx seguono la IEC 60079-0:


Ex db IIC T4 Gb
│  │  │   │  │
│  │  │   │  └─ EPL
│  │  │   └──── Classe di temperatura
│  │  └──────── Gruppo gas
│  └─────────── Tipo di protezione con suffisso di livello
└────────────── Simbolo Ex

Le marcature IECEx non includono il marchio CE, il numero di gruppo o la categoria. Il numero del certificato IECEx fornisce la tracciabilità:


IECEx BAS 12.0034X
│     │   │
│     │   └─ Numero del certificato + revisione (X = condizioni speciali)
│     └───── Codice ExCB (BAS = Baseefa, UK)
└─────────── Identificativo dello schema IECEx

Marcature combinate

Apparecchiature con più metodi di protezione:


Ex db eb IIC T4 Gb

Corpo principale: a prova di esplosione (db). Vano morsetti: sicurezza aumentata (eb).


Ex db [ib Ga] IIC T4 Gb

Custodia a prova di esplosione (Gb) contenente circuiti a sicurezza intrinseca (Ga). Le parentesi indicano i circuiti IS interni.


Ex db IIC T6...T4 Gb

La classe di temperatura varia con la temperatura ambiente. T6 alla temperatura ambiente standard, T4 alla temperatura ambiente elevata.

Doppia marcatura gas e polvere

Apparecchiatura certificata sia per gas che per polvere:


II 2G  Ex db IIC T4 Gb
II 2D  Ex tb IIIC T125°C Db

Entrambe le righe appaiono sulla targhetta. L'apparecchiatura può essere utilizzata in Zona 1 gas o Zona 21 polvere.

Lettura di esempi reali

Esempio 1: Cassetta di giunzione


CE 0539 ⟨Ex⟩ II 2G Ex eb IIC T5 Gb
  • ATEX Gruppo II, Categoria 2G (Zona 1 gas)
  • Sicurezza aumentata (Ex eb)
  • Tutti i gruppi gas (IIC)
  • T5 (temp. max. superficiale 100°C)
  • EPL Gb

Esempio 2: Proiettore LED


CE 0539 ⟨Ex⟩ II 2G Ex db eb IIC T4 Gb
                    II 2D Ex tb IIIC T135°C Db
  • Doppia classificazione: gas (Zona 1) e polvere (Zona 21)
  • Corpo a prova di esplosione con cassetta morsetti a sicurezza aumentata
  • Tutti i gruppi gas (IIC), tutti i gruppi polvere (IIIC)
  • Gas: T4 (135°C max), Polvere: 135°C max

Esempio 3: Custodia iPhone certificata ATEX (Xshielder)


⟨Ex⟩ II 2G Ex db IIC T4 Gb
     II 2D Ex tb IIIC T135°C Db
  • Zona 1 gas + Zona 21 polvere
  • Custodia a prova di esplosione + protezione dalla polvere
  • Copre tutti i gruppi gas e polvere
  • Classificazione IP65, compatibile con ricarica wireless

Esempio 4: Trasmettitore a sicurezza intrinseca


CE 0539 ⟨Ex⟩ II 1G Ex ia IIC T4 Ga
  • Categoria 1G — idoneo per Zona 0
  • Sicurezza intrinseca (livello massimo, Ex ia)
  • Tutti i gruppi gas (IIC)
  • T4 (135°C max)
  • EPL Ga (protezione molto alta)

Esempio 5: Motore per Zona 2


CE 0539 ⟨Ex⟩ II 3G Ex nA IIB T3 Gc
  • Categoria 3G — solo Zona 2
  • Non scintillante (Ex nA)
  • Solo gruppi gas IIA e IIB (non IIC)
  • T3 (200°C max)
  • EPL Gc

Marcature aggiuntive sulla targhetta

Oltre alla marcatura Ex, le targhette delle apparecchiature includono:

MarcaturaSignificato
Ta o TambIntervallo di temperatura ambiente (ad es., −40°C a +60°C)
IP66 / IP68Grado di protezione IP
XCondizioni speciali d'uso (vedi certificato)
UCertificato di componente (non autonomo)
N. certificatoRiferimento al certificato di prova completo
Numero di serieIdentificativo univoco del dispositivo per la tracciabilità
Anno di fabbricazioneRichiesto per l'iter di verifica

Il suffisso "X" è importante — significa che ci sono condizioni speciali documentate nel certificato che devono essere rispettate durante l'installazione e l'uso. Recuperare e leggere sempre il certificato quando si vede "X".

Marcature NEC/CEC (Nord America)

La marcatura nordamericana segue un formato diverso:


Class I, Division 1, Groups B, C, D, T4

Oppure nel sistema a zone:


Class I, Zone 1, AEx db IIC T4 Gb

Il prefisso "A" prima di "Ex" indica che l'apparecchiatura è certificata secondo le norme nordamericane (NEC/CEC) utilizzando il sistema a zone.
