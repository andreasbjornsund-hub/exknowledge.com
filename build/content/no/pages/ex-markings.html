Hvordan lese Ex-merkinger på utstyr

Hvert sertifisert Ex-utstyr har en merking som koder vernetype, gass-/støvgruppe, temperaturklasse og utstyrsvernenivå. Å forstå disse merkingene er avgjørende for å velge, installere og inspisere utstyr.

ATEX-merkingsstruktur

ATEX-utstyr har en strukturert merking i henhold til Direktiv 2014/34/EU:


CE 0539  ⟨Ex⟩  II 2G  Ex db IIC T4 Gb
│   │     │     │  │    │  │  │   │  │
│   │     │     │  │    │  │  │   │  └─ EPL (Gb = Høy, Sone 1)
│   │     │     │  │    │  │  │   └──── Temperaturklasse (T4 = maks 135 °C)
│   │     │     │  │    │  │  └──────── Gassgruppe (IIC = hydrogen/acetylen)
│   │     │     │  │    │  └─────────── Vernetype (db = flammesikker, EPL b)
│   │     │     │  │    └────────────── Ex-symbol (eksplosjonsbeskyttet)
│   │     │     │  └─────────────────── Kategori (2G = Kategori 2, Gass)
│   │     │     └────────────────────── Utstyrsgruppe (II = overflateindustri)
│   │     └──────────────────────────── ATEX sekskantet «Ex»-symbol
│   └────────────────────────────────── Bemyndiget organ-nummer
└────────────────────────────────────── CE-merke

IECEx-merkingsstruktur

IECEx-merkinger følger IEC 60079-0:


Ex db IIC T4 Gb
│  │  │   │  │
│  │  │   │  └─ EPL
│  │  │   └──── Temperaturklasse
│  │  └──────── Gassgruppe
│  └─────────── Vernetype med nivåsuffiks
└────────────── Ex-symbol

IECEx-merkinger inkluderer ikke CE-merket, gruppenummer eller kategori. I stedet gir IECEx-sertifikatnummeret sporbarhet:


IECEx BAS 12.0034X
│     │   │
│     │   └─ Sertifikatnummer + revisjon (X = spesielle betingelser)
│     └───── ExCB-kode (BAS = Baseefa, UK)
└─────────── IECEx-ordningsidentifikator

Kombinerte merkinger

Utstyr med flere vernemetoder:


Ex db eb IIC T4 Gb

Hoveddel: flammesikker (db). Terminalkammer: økt sikkerhet (eb).


Ex db [ib Ga] IIC T4 Gb

Flammesikker innkapsling (Gb) som inneholder egensikre kretser (Ga). Klammeparentesene angir IS-kretsene inni.


Ex db IIC T6...T4 Gb

Temperaturklassen varierer med omgivelsestemperatur. T6 ved standard omgivelse, T4 ved forhøyet omgivelse.

Dobbel gass- og støvmerking

Utstyr sertifisert for både gass og støv:


II 2G  Ex db IIC T4 Gb
II 2D  Ex tb IIIC T125°C Db

Begge linjene vises på merkeplata. Utstyret kan brukes i gassone 1 eller støvsone 21.

Lesing av virkelige eksempler

Eksempel 1: Koblingsboks


CE 0539 ⟨Ex⟩ II 2G Ex eb IIC T5 Gb
  • ATEX Gruppe II, Kategori 2G (Sone 1 gass)
  • Økt sikkerhet (Ex eb)
  • Alle gassgrupper (IIC)
  • T5 (maks 100 °C overflate)
  • EPL Gb

Eksempel 2: LED-flomlys


CE 0539 ⟨Ex⟩ II 2G Ex db eb IIC T4 Gb
                    II 2D Ex tb IIIC T135°C Db
  • Dobbelt godkjent: gass (Sone 1) og støv (Sone 21)
  • Flammesikker kropp med økt sikkerhet-terminalboks
  • Alle gassgrupper (IIC), alle støvgrupper (IIIC)
  • Gass: T4 (135 °C maks), Støv: 135 °C maks

Eksempel 3: ATEX-sertifisert iPhone-deksel (Xshielder)


⟨Ex⟩ II 2G Ex db IIC T4 Gb
     II 2D Ex tb IIIC T135°C Db
  • Sone 1 gass + Sone 21 støv
  • Flammesikker + støvbeskyttet innkapsling
  • Dekker alle gassgrupper og støvgrupper
  • IP65-gradert, kompatibel med trådløs lading

Eksempel 4: Egensikker transmitter


CE 0539 ⟨Ex⟩ II 1G Ex ia IIC T4 Ga
  • Kategori 1G — egnet for Sone 0
  • Egensikker (høyeste nivå, Ex ia)
  • Alle gassgrupper (IIC)
  • T4 (135 °C maks)
  • EPL Ga (svært høy beskyttelse)

Eksempel 5: Sone 2-motor


CE 0539 ⟨Ex⟩ II 3G Ex nA IIB T3 Gc
  • Kategori 3G — kun Sone 2
  • Ikke-gnistdannende (Ex nA)
  • Gassgrupper IIA og IIB kun (ikke IIC)
  • T3 (200 °C maks)
  • EPL Gc

Tilleggsmerkinger på merkeplata

I tillegg til Ex-merkinga inkluderer utstyrsetiketter:

Merking Betydning
Ta eller TambOmgivelsestemperaturområde (f.eks. −40 °C til +60 °C)
IP66 / IP68Kapslingsgrad
XSpesielle bruksbetingelser (se sertifikat)
UKomponentsertifikat (ikke frittstående)
Sertifikatnr.Referanse til fullstendig testsertifikat
SerienummerUnik enhetsidentifikator for sporbarhet
ProduksjonsårPåkrevd for revisjonsspor

"X"-suffikset er viktig — det betyr at det finnes spesielle betingelser dokumentert i sertifikatet som må overholdes under installasjon og bruk. Hent og les alltid sertifikatet når du ser "X".

NEC/CEC-merkinger (Nord-Amerika)

Nordamerikansk merking følger et annet format:


Class I, Division 1, Groups B, C, D, T4

Eller i sonesystemet:


Class I, Zone 1, AEx db IIC T4 Gb

"A"-prefikset foran "Ex" angir at utstyret er sertifisert etter nordamerikanske standarder (NEC/CEC) med sonesystemet.
