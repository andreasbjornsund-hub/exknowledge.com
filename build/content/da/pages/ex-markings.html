Sådan læses Ex-mærkning på udstyr

Hvert certificeret Ex-udstyr bærer en mærkning, der koder dets beskyttelsestype, gas-/støvgruppe, temperaturklasse og udstyrets beskyttelsesniveau. At forstå disse mærkninger er afgørende for valg, installation og inspektion af udstyr.

ATEX-mærkningsstruktur

ATEX-udstyr bærer en struktureret mærkning iht. direktiv 2014/34/EU:


CE 0539  ⟨Ex⟩  II 2G  Ex db IIC T4 Gb
│   │     │     │  │    │  │  │   │  │
│   │     │     │  │    │  │  │   │  └─ EPL (Gb = Højt, Zone 1)
│   │     │     │  │    │  │  │   └──── Temperaturklasse (T4 = maks. 135°C)
│   │     │     │  │    │  │  └──────── Gasgruppe (IIC = hydrogen/acetylen)
│   │     │     │  │    │  └─────────── Beskyttelsestype (db = eksplosionssikker, EPL b)
│   │     │     │  │    └────────────── Ex-symbol (eksplosionsbeskyttet)
│   │     │     │  └─────────────────── Kategori (2G = Kategori 2, Gas)
│   │     │     └────────────────────── Udstyrsgruppe (II = overfladeindustrier)
│   │     └──────────────────────────── ATEX sekskant "Ex"-symbol
│   └────────────────────────────────── Bemyndiget organs nummer
└────────────────────────────────────── CE-mærke

ATEX-udstyrsgrupper

Gruppe Anvendelse
I Minedrift (grubegas)
II Overfladeindustrier — gas/damp
III Overfladeindustrier — støv

ATEX-kategorier

Mærkning Kategori EPL Zoner
1G Kat. 1 Gas Ga 0, 1, 2
2G Kat. 2 Gas Gb 1, 2
3G Kat. 3 Gas Gc 2
1D Kat. 1 Støv Da 20, 21, 22
2D Kat. 2 Støv Db 21, 22
3D Kat. 3 Støv Dc 22

IECEx-mærkningsstruktur

IECEx-mærkninger følger IEC 60079-0:


Ex db IIC T4 Gb
│  │  │   │  │
│  │  │   │  └─ EPL
│  │  │   └──── Temperaturklasse
│  │  └──────── Gasgruppe
│  └─────────── Beskyttelsestype med niveausuffiks
└────────────── Ex-symbol

IECEx-mærkninger inkluderer ikke CE-mærket, gruppenummer eller kategori. I stedet giver IECEx-certifikatnummeret sporbarhed:


IECEx BAS 12.0034X
│     │   │
│     │   └─ Certifikatnummer + revision (X = særlige betingelser)
│     └───── ExCB-kode (BAS = Baseefa, UK)
└─────────── IECEx-ordnings-identifikator

Aflæsning af beskyttelsestypen

Beskyttelsestypen bruger præfikset "Ex" med to bogstaver efterfulgt af metodebogstavet og niveausuffikset:

Mærkning Metode Niveau Betydning
Ex ia Egensikkerhed a Zone 0-egnet
Ex ib Egensikkerhed b Zone 1-egnet
Ex ic Egensikkerhed c Kun Zone 2
Ex db Eksplosionssikker b Zone 1-egnet
Ex dc Eksplosionssikker c Kun Zone 2
Ex eb Øget sikkerhed b Zone 1-egnet
Ex ec Øget sikkerhed c Kun Zone 2
Ex ma Indkapsling a Zone 0-egnet
Ex mb Indkapsling b Zone 1-egnet
Ex mc Indkapsling c Kun Zone 2
Ex px Overtryk x Zone 1 → ikke-farligt
Ex py Overtryk y Zone 1 → Zone 2
Ex pz Overtryk z Zone 2 → ikke-farligt
Ex nA Ikke-gnistdannende — Kun Zone 2
Ex nC Lukkede/forseglede kontakter — Kun Zone 2
Ex nR Begrænset vejrtrækning — Kun Zone 2
Ex ta Støvindkapsling a Zone 20-egnet
Ex tb Støvindkapsling b Zone 21-egnet
Ex tc Støvindkapsling c Kun Zone 22
Ex o Olieneddykning — Zone 1-egnet
Ex q Pulverfyldning — Zone 1-egnet

Kombinerede mærkninger

Udstyr med flere beskyttelsesmetoder:


Ex db eb IIC T4 Gb

Hovedkrop: eksplosionssikker (db). Klemrum: øget sikkerhed (eb).


Ex db [ib Ga] IIC T4 Gb

Eksplosionssikker indkapsling (Gb) indeholdende egensikre kredsløb (Ga). Klammerne angiver IS-kredsløbene indeni.


Ex db IIC T6...T4 Gb

Temperaturklassen varierer med omgivelsestemperaturen. T6 ved standard omgivelsestemperatur, T4 ved forhøjet omgivelsestemperatur.

Dobbelt gas- og støvmærkning

Udstyr certificeret til både gas og støv:


II 2G  Ex db IIC T4 Gb
II 2D  Ex tb IIIC T125°C Db

Begge linjer fremgår af typeskiltet. Udstyret kan bruges i gas Zone 1 eller støv Zone 21.

Aflæsning af eksempler fra virkeligheden

Eksempel 1: Klemkasse


CE 0539 ⟨Ex⟩ II 2G Ex eb IIC T5 Gb
  • ATEX Gruppe II, Kategori 2G (Zone 1 gas)
  • Øget sikkerhed (Ex eb)
  • Alle gasgrupper (IIC)
  • T5 (maks. 100°C overflade)
  • EPL Gb

Eksempel 2: LED-projektor


CE 0539 ⟨Ex⟩ II 2G Ex db eb IIC T4 Gb
                    II 2D Ex tb IIIC T135°C Db
  • Dobbeltklassificeret: gas (Zone 1) og støv (Zone 21)
  • Eksplosionssikker krop med øget sikkerhed klemkasse
  • Alle gasgrupper (IIC), alle støvgrupper (IIIC)
  • Gas: T4 (135°C maks.), Støv: 135°C maks.

Eksempel 3: ATEX-certificeret iPhone-cover (Xshielder)


⟨Ex⟩ II 2G Ex db IIC T4 Gb
     II 2D Ex tb IIIC T135°C Db
  • Zone 1 gas + Zone 21 støv
  • Eksplosionssikker + støvbeskyttet indkapsling
  • Dækker alle gasgrupper og støvgrupper
  • IP65-klassificeret, trådløs opladning kompatibel

Eksempel 4: Egensikker transmitter


CE 0539 ⟨Ex⟩ II 1G Ex ia IIC T4 Ga
  • Kategori 1G — egnet til Zone 0
  • Egensikker (højeste niveau, Ex ia)
  • Alle gasgrupper (IIC)
  • T4 (135°C maks.)
  • EPL Ga (meget høj beskyttelse)

Eksempel 5: Zone 2-motor


CE 0539 ⟨Ex⟩ II 3G Ex nA IIB T3 Gc
  • Kategori 3G — kun Zone 2
  • Ikke-gnistdannende (Ex nA)
  • Kun gasgrupper IIA og IIB (ikke IIC)
  • T3 (200°C maks.)
  • EPL Gc

Yderligere mærkninger på typeskiltet

Ud over Ex-mærkningen inkluderer udstyrets etiketter:

Mærkning Betydning
Ta eller Tamb Omgivelsestemperaturområde (f.eks. −40°C til +60°C)
IP66 / IP68 IP-klassifikation
X Særlige brugsbetingelser (se certifikat)
U Komponentcertifikat (ikke stand-alone)
Certifikat nr. Reference til det fulde testcertifikat
Serienummer Unikt enheds-ID for sporbarhed
Produktionsår Krævet for auditspor

Suffikset "X" er vigtigt — det betyder, at der er særlige betingelser dokumenteret i certifikatet, som skal overholdes under installation og brug. Hent og læs altid certifikatet, når du ser "X".

NEC/CEC-mærkninger (Nordamerika)

Nordamerikansk mærkning følger et andet format:


Class I, Division 1, Groups B, C, D, T4

Eller i zonesystemet:


Class I, Zone 1, AEx db IIC T4 Gb

Præfikset "A" før "Ex" angiver, at udstyret er certificeret til nordamerikanske standarder (NEC/CEC) ved brug af zonesystemet.
