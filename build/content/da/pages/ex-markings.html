Sådan læses Ex-mærkninger på udstyr

Hvert certificeret Ex-apparat bærer en mærkning, der indkoder beskyttelsestype, gas-/støvgruppe, temperaturklasse og udstyrets beskyttelsesniveau. Forståelse af disse mærkninger er afgørende for valg, installation og inspektion af udstyr.

ATEX-mærkningsstruktur

ATEX-udstyr bærer en struktureret mærkning iht. Direktiv 2014/34/EU:


CE 0539  ⟨Ex⟩  II 2G  Ex db IIC T4 Gb
│   │     │     │  │    │  │  │   │  │
│   │     │     │  │    │  │  │   │  └─ EPL (Gb = Høj, Zone 1)
│   │     │     │  │    │  │  │   └──── Temperaturklasse (T4 = maks. 135 °C)
│   │     │     │  │    │  │  └──────── Gasgruppe (IIC = brint/acetylen)
│   │     │     │  │    │  └─────────── Beskyttelsestype (db = flammesikkert, EPL b)
│   │     │     │  │    └────────────── Ex-symbol (eksplosionsbeskyttet)
│   │     │     │  └─────────────────── Kategori (2G = Kategori 2, Gas)
│   │     │     └────────────────────── Udstyrsgruppe (II = overfladeindustri)
│   │     └──────────────────────────── ATEX sekskantet "Ex"-symbol
│   └────────────────────────────────── Bemyndiget organs nummer
└────────────────────────────────────── CE-mærke

IECEx-mærkningsstruktur

IECEx-mærkninger følger IEC 60079-0:


Ex db IIC T4 Gb
│  │  │   │  │
│  │  │   │  └─ EPL
│  │  │   └──── Temperaturklasse
│  │  └──────── Gasgruppe
│  └─────────── Beskyttelsestype med niveausuffiks
└────────────── Ex-symbol

IECEx-mærkninger inkluderer ikke CE-mærket, gruppenummer eller kategori. I stedet giver IECEx-certifikatnummeret sporbarhed:


IECEx BAS 12.0034X
│     │   │
│     │   └─ Certifikatnummer + revision (X = særlige betingelser)
│     └───── ExCB-kode (BAS = Baseefa, UK)
└─────────── IECEx-ordningsidentifikator

Kombinerede mærkninger

Udstyr med flere beskyttelsesmetoder:


Ex db eb IIC T4 Gb

Hoveddel: flammesikkert (db). Klemmekammer: forhøjet sikkerhed (eb).


Ex db [ib Ga] IIC T4 Gb

Flammesikkert kabinet (Gb) med egensikre kredsløb (Ga) indeni. Parenteserne angiver de egensikre kredsløb indeni.


Ex db IIC T6...T4 Gb

Temperaturklassen varierer med omgivelsestemperaturen. T6 ved standard omgivelser, T4 ved forhøjede omgivelser.

Dobbelt gas- og støvmærkning

Udstyr certificeret til både gas og støv:


II 2G  Ex db IIC T4 Gb
II 2D  Ex tb IIIC T125°C Db

Begge linjer optræder på typeskiltet. Udstyret kan bruges i gas Zone 1 eller støv Zone 21.

Aflæsning af praktiske eksempler

Eksempel 1: Samledåse


CE 0539 ⟨Ex⟩ II 2G Ex eb IIC T5 Gb
  • ATEX Gruppe II, Kategori 2G (Zone 1 gas)
  • Forhøjet sikkerhed (Ex eb)
  • Alle gasgrupper (IIC)
  • T5 (maks. 100 °C overflade)
  • EPL Gb

Eksempel 2: LED-projektør


CE 0539 ⟨Ex⟩ II 2G Ex db eb IIC T4 Gb
                    II 2D Ex tb IIIC T135°C Db
  • Dobbeltklassificeret: gas (Zone 1) og støv (Zone 21)
  • Flammesikkert hus med forhøjet sikkerhed-klemmeboks
  • Alle gasgrupper (IIC), alle støvgrupper (IIIC)
  • Gas: T4 (135 °C maks.), Støv: 135 °C maks.

Eksempel 3: ATEX-certificeret iPhone-etui (Xshielder)


⟨Ex⟩ II 2G Ex db IIC T4 Gb
     II 2D Ex tb IIIC T135°C Db
  • Zone 1 gas + Zone 21 støv
  • Flammesikkert + støvbeskyttet kabinet
  • Dækker alle gasgrupper og støvgrupper
  • IP65-klassificeret, trådløs opladning kompatibel

Eksempel 4: Egensikker transmitter


CE 0539 ⟨Ex⟩ II 1G Ex ia IIC T4 Ga
  • Kategori 1G — egnet til Zone 0
  • Egensikker (højeste niveau, Ex ia)
  • Alle gasgrupper (IIC)
  • T4 (135 °C maks.)
  • EPL Ga (meget høj beskyttelse)

Eksempel 5: Zone 2-motor


CE 0539 ⟨Ex⟩ II 3G Ex nA IIB T3 Gc
  • Kategori 3G — kun Zone 2
  • Gnistfri (Ex nA)
  • Gasgrupper IIA og IIB kun (ikke IIC)
  • T3 (200 °C maks.)
  • EPL Gc

"X"-suffikset er vigtigt — det betyder, at der er særlige betingelser dokumenteret i certifikatet, som skal overholdes under installation og brug. Hent og læs altid certifikatet, når du ser "X".

NEC/CEC-mærkninger (Nordamerika)

Nordamerikansk mærkning følger et andet format:


Class I, Division 1, Groups B, C, D, T4

Eller i zonesystemet:


Class I, Zone 1, AEx db IIC T4 Gb

"A"-præfikset foran "Ex" angiver, at udstyret er certificeret efter nordamerikanske standarder (NEC/CEC) med brug af zonesystemet.
