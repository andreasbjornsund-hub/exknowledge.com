Kuinka lukea Ex-merkinnät laitteissa

Jokaisessa sertifioidussa Ex-laitteessa on merkintä, joka koodaa sen suojaustyypin, kaasu-/pölyryhmän, lämpötilaluokan ja laitteiden suojaustason. Näiden merkintöjen ymmärtäminen on välttämätöntä laitteiden valinnassa, asennuksessa ja tarkastamisessa.

ATEX-merkintärakenne

ATEX-laitteissa on direktiivin 2014/34/EU mukainen strukturoitu merkintä:


CE 0539 ⟨Ex⟩ II 2G Ex db IIC T4 Gb
│ │ │ │ │ │ │ │ │ │
│ │ │ │ │ │ │ │ │ └─ EPL (Gb = korkea, vyöhyke 1)
│ │ │ │ │ │ │ │ └──── Lämpötilaluokka (T4 = max 135°C)
│ │ │ │ │ │ │ └──────── Kaasuryhmä (IIC = vety/asetyleeni)
│ │ │ │ │ │ └─────────── Suojaustyyppi (db = tulenkestävä, EPL b)
│ │ │ │ │ └────────────── Ex-symboli (räjähdyssuojattu)
│ │ │ │ └─────────────────── Luokka (2G = luokka 2, kaasu)
│ │ │ └───────────────────── Laiteryhmä (II = pintateollisuus)
│ │ └───────────────────────────────────
│ └───────────────────────────────────
└───────────────────────────────── CE-merkki

ATEX-laiteryhmät

Ryhmä Hakemus
I Kaivostoiminta (polttoamp)
II Pintateollisuus – kaasu/höyry
III Pintateollisuus — pöly

ATEX-luokat

Merkintä Luokka EPL Alueet
1G Cat 1 Kaasu Ga 0, 1, 2

IECEx-merkintärakenne

IECEx-merkinnät noudattavat standardia IEC 60079-0:


Ex db IIC T4 Gb
│ │ │ │ │
│ │ │ │ └─ EPL
│ │ │ └──── Lämpötilaluokka
│ │ └──────── Kaasuryhmä
│ └─────────── Suojaustyyppi tasoliitteellä
└────────────── Ex-symboli

IECEx-merkinnät eivät sisällä CE-merkkiä, ryhmänumeroa tai luokkaa. Sen sijaan IECEx-sertifikaatin numero tarjoaa jäljitettävyyden:


IECEx BAS 12.0034X
│ │ │
│ │ └─ Sertifikaatin numero + versio (X = erityisehdot)
│ └───── ExCB-koodi (BAS = Baseefa, UK)
└─────────── IECEx-järjestelmän tunniste

Suojaustyypin lukeminen

Suojaustyyppi käyttää kaksikirjaimista "Ex"-etuliitettä, jota seuraa metodikirjain ja tasoliite:

Merkintä Menetelmä Taso Merkitys
Ex ia Luontainen turvallisuus a Alue 0 kykenee
Ex ib Luontainen turvallisuus b Alue 1 pystyy
Ex ic Luontainen turvallisuus c Vain vyöhyke 2
Ex db Tulenkestävä b Alue 1 pystyy
Ex dc Tulenkestävä c Vain vyöhyke 2
Ex eb Lisääntynyt turvallisuus b Alue 1 pystyy
Ex ec Lisääntynyt turvallisuus c Vain vyöhyke 2
Ex ma Kapselointi a Alue 0 kykenee
Ex mb Kapselointi b Alue 1 pystyy
Ex mc Kapselointi c Vain vyöhyke 2
Ex px Paine x Alue 1 → vaaraton
Ex py Paine y Alue 1 → vyöhyke 2
Ex pz Paine z Alue 2 → ei-haz
Ex nA Kipinöimätön — Vain vyöhyke 2
Ex nC Suljetut/suljetut koskettimet — Vain vyöhyke 2
Ex nR Rajoitettu hengitys — Vain vyöhyke 2
Ex ta Pölykotelo a Alue 20 kykenee
Ex tb Pölykotelo b Vyöhyke 21 pystyy
Ex tc Pölykotelo c Vain vyöhyke 22
Ex o Öljyupotus — Alue 1 pystyy
Ex q Jauhetäyte — Alue 1 pystyy

Yhdistetyt merkinnät

Laitteet, joissa on useita suojausmenetelmiä:


Ex db eb IIC T4 Gb

Päärunko: tulenkestävä (db). Liitäntälokero: parempi turvallisuus (eb).


Ex db [ib Ga] IIC T4 Gb

Tulenkestävä kotelo (Gb), joka sisältää luonnostaan ​​vaarattomia piirejä (Ga). Sulut osoittavat sisällä olevat IS-piirit.


Ex db IIC T6...T4 Gb

Lämpötilaluokka vaihtelee ympäristön lämpötilan mukaan. T6 normaalissa ympäristössä, T4 korotetussa ympäristössä.

Kaksoiskaasu- ja pölymerkintä

Sekä kaasulle että pölylle sertifioidut laitteet:


II 2G Ex db IIC T4 Gb
II 2D Ex tb IIIC T125°C Db

Molemmat rivit näkyvät tyyppikilvessä. Laitetta voidaan käyttää kaasuvyöhykkeellä 1 tai pölyvyöhykkeellä 21.

Tosimaailman esimerkkien lukeminen

Esimerkki 1: Kytkentärasia


CE 0539 ⟨Ex⟩ II 2G Ex eb IIC T5 Gb
  • ATEX Group II, luokka 2G (vyöhykkeen 1 kaasu)
  • Lisääntynyt turvallisuus (Ex eb)
  • Kaikki kaasuryhmät (IIC)
  • T5 (max 100°C pinta)
  • EPL Gb

Esimerkki 2: LED-valonheitin


CE 0539 ⟨Ex⟩ II 2G Ex db eb IIC T4 Gb
                    II 2D Ex tb IIIC T135°C Db
  • Kaksoisluokitus: kaasu (vyöhyke 1) ja pöly (vyöhyke 21)
  • Tulenkestävä runko parannetulla turvaliitinkotelolla
  • Kaikki kaasuryhmät (IIC), kaikki pölyryhmät (IIIC)
  • Kaasu: T4 (135°C max), pöly: 135°C max

Esimerkki 3: ATEX-sertifioitu iPhone-kotelo (Xshielder)


⟨Ex⟩ II 2G Ex db IIC T4 Gb
     II 2D Ex tb IIIC T135°C Db
  • Vyöhykkeen 1 kaasu + vyöhykkeen 21 pöly
  • Tulenkestävä + pölyltä suojattu kotelo
  • Kattaa kaikki kaasu- ja pölyryhmät
  • IP65-luokiteltu, langaton lataus yhteensopiva

Esimerkki 4: Luonnostaan ​​vaaraton lähetin


CE 0539 ⟨Ex⟩ II 1G Ex ia IIC T4 Ga
  • Luokka 1G — sopii vyöhykkeelle 0
  • Luonnostaan ​​turvallinen (korkein taso, Ex ia)
  • Kaikki kaasuryhmät (IIC)
  • T4 (135°C max)
  • EPL Ga (erittäin korkea suojaus)

Esimerkki 5: Vyöhykkeen 2 moottori


CE 0539 ⟨Ex⟩ II 3G Ex nA IIB T3 Gc
  • Luokka 3G — vain vyöhyke 2
  • Kipinöimätön (Ex nA)
  • Vain kaasuryhmät IIA ja IIB (ei IIC)
  • T3 (200°C max)
  • EPL Gc

Lisämerkinnät nimikilvessä

Ex-merkinnän lisäksi laitetarrat sisältävät:

Merkintä Merkitys
Ta tai Tamb Ympäristön lämpötila-alue (esim. −40°C - +60°C)
IP66 / IP68 Sisäänpääsyn suojausluokitus
X Erityiset käyttöehdot (katso todistus)
U Komponenttitodistus (ei erillinen)
Todistus nro Viittaus täydelliseen testitodistukseen
Sarjanumero Ainutlaatuinen laitetunniste jäljitettävyyttä varten
Valmistusvuosi Vaaditaan kirjausketjua varten

The "X" pääte on tärkeä – se tarkoittaa, että sertifikaatissa on dokumentoitu erityisehtoja, joita on noudatettava asennuksen ja käytön aikana. Hae ja lue varmenne aina, kun näet "X".

NEC/CEC-merkinnät (Pohjois-Amerikka)

Pohjois-Amerikan merkintä noudattaa eri muotoa:


Luokka I, Division 1, ryhmät B, C, D, T4

Tai Zone-järjestelmässä:


Luokka I, vyöhyke 1, AEx db IIC T4 Gb

"A"-etuliite ennen "Ex" tarkoittaa, että laite on sertifioitu Pohjois-Amerikan standardien (NEC/CEC) mukaisesti käyttämällä Zone-järjestelmää.

Aiheeseen liittyvät tiedostot

  • Suojausmenetelmät — yksityiskohtainen selvitys kustakin menetelmästä
  • Kaasuryhmät — mitä IIA/IIB/IIC tarkoittaa
  • Lämpötilaluokat — mitä T1–T6 tarkoittaa
  • EPL — mitä Ga/Gb/Gc tarkoittaa
  • Standardit — mikä standardi säätelee kutakin merkintäelementtiä
  • Sertifiointi — miten todistukset ja merkinnät myönnetään
