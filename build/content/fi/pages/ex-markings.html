Ex-merkintöjen tulkinta

Jokainen sertifioitu Ex-laite on merkitty tavalla, joka koodaa sen suojaustyypin, kaasu-/pölyryhmän, lämpötilaluokan ja laitesuojaustason. Näiden merkintöjen ymmärtäminen on välttämätöntä laitteiden valinnassa, asennuksessa ja tarkastuksessa.

ATEX-merkinnän rakenne

ATEX-laitteet ovat merkitty direktiivin 2014/34/EU mukaisesti:


CE 0539  ⟨Ex⟩  II 2G  Ex db IIC T4 Gb
│   │     │     │  │    │  │  │   │  │
│   │     │     │  │    │  │  │   │  └─ EPL (Gb = Korkea, vyöhyke 1)
│   │     │     │  │    │  │  │   └──── Lämpötilaluokka (T4 = enint. 135 °C)
│   │     │     │  │    │  │  └──────── Kaasuryhmä (IIC = vety/asetyleeni)
│   │     │     │  │    │  └─────────── Suojaustyyppi (db = räjähdyspaineen kestävä, taso b)
│   │     │     │  │    └────────────── Ex-symboli (räjähdyssuojattu)
│   │     │     │  └─────────────────── Kategoria (2G = Kategoria 2, Kaasu)
│   │     │     └────────────────────── Laiteryhmä (II = pintateollisuus)
│   │     └──────────────────────────── ATEX:n kuusikulmainen "Ex"-symboli
│   └────────────────────────────────── Ilmoitetun laitoksen numero
└────────────────────────────────────── CE-merkki

IECEx-merkinnän rakenne

IECEx-merkinnät noudattavat standardia IEC 60079-0:


Ex db IIC T4 Gb
│  │  │   │  │
│  │  │   │  └─ EPL
│  │  │   └──── Lämpötilaluokka
│  │  └──────── Kaasuryhmä
│  └─────────── Suojaustyyppi tasomerkinnällä
└────────────── Ex-symboli

IECEx-merkinnät eivät sisällä CE-merkkiä, ryhmänumeroa tai kategoriaa. Sen sijaan IECEx-sertifikaattinumero tarjoaa jäljitettävyyden:


IECEx BAS 12.0034X
│     │   │
│     │   └─ Sertifikaattinumero + revisio (X = erityisehdot)
│     └───── ExCB-koodi (BAS = Baseefa, UK)
└─────────── IECEx-järjestelmän tunniste

Yhdistetyt merkinnät

Laitteissa, joissa on useita suojausmenetelmiä:


Ex db eb IIC T4 Gb

Runko: räjähdyspaineen kestävä (db). Liitintila: varmennettu rakenne (eb).


Ex db [ib Ga] IIC T4 Gb

Räjähdyspaineen kestävä kotelo (Gb), joka sisältää luontaisesti turvallisia piirejä (Ga). Sulkeet osoittavat IS-piirit sisällä.


Ex db IIC T6...T4 Gb

Lämpötilaluokka vaihtelee ympäristölämpötilan mukaan. T6 normaalissa ympäristössä, T4 kohotetussa ympäristölämpötilassa.

Kaasu- ja pölymerkintä samassa laitteessa

Laite, joka on sertifioitu sekä kaasulle että pölylle:


II 2G  Ex db IIC T4 Gb
II 2D  Ex tb IIIC T125°C Db

Molemmat rivit näkyvät tyyppikilvessä. Laitetta voidaan käyttää kaasuvyöhykkeessä 1 tai pölyvyöhykkeessä 21.

Todellisten esimerkkien tulkinta

Esimerkki 1: Kytkentärasia


CE 0539 ⟨Ex⟩ II 2G Ex eb IIC T5 Gb
  • ATEX Ryhmä II, Kategoria 2G (vyöhyke 1, kaasu)
  • Varmennettu rakenne (Ex eb)
  • Kaikki kaasuryhmät (IIC)
  • T5 (enintään 100 °C pintalämpötila)
  • EPL Gb

Esimerkki 2: LED-valonheitin


CE 0539 ⟨Ex⟩ II 2G Ex db eb IIC T4 Gb
                    II 2D Ex tb IIIC T135°C Db
  • Kaksoisluokiteltu: kaasu (vyöhyke 1) ja pöly (vyöhyke 21)
  • Räjähdyspaineen kestävä runko ja varmennetun rakenteen liitinrasia
  • Kaikki kaasuryhmät (IIC), kaikki pölyryhmät (IIIC)
  • Kaasu: T4 (135 °C enint.), Pöly: 135 °C enint.

Esimerkki 3: ATEX-sertifioitu iPhone-kotelo (Xshielder)


⟨Ex⟩ II 2G Ex db IIC T4 Gb
     II 2D Ex tb IIIC T135°C Db
  • Vyöhyke 1, kaasu + vyöhyke 21, pöly
  • Räjähdyspaineen kestävä + pölysuojattu kotelo
  • Kattaa kaikki kaasu- ja pölyryhmät
  • IP65-luokiteltu, langaton lataus yhteensopiva

Esimerkki 4: Luontaisesti turvallinen lähetin


CE 0539 ⟨Ex⟩ II 1G Ex ia IIC T4 Ga
  • Kategoria 1G — soveltuu vyöhykkeelle 0
  • Luontaisesti turvallinen (korkein taso, Ex ia)
  • Kaikki kaasuryhmät (IIC)
  • T4 (135 °C enint.)
  • EPL Ga (erittäin korkea suojaus)

Esimerkki 5: Vyöhyke 2 -moottori


CE 0539 ⟨Ex⟩ II 3G Ex nA IIB T3 Gc
  • Kategoria 3G — vain vyöhyke 2
  • Kipinöimätön (Ex nA)
  • Kaasuryhmät IIA ja IIB (ei IIC)
  • T3 (200 °C enint.)
  • EPL Gc

"X"-pääte on tärkeä — se tarkoittaa, että sertifikaatissa on erityisehtoja, joita on noudatettava asennuksessa ja käytössä. Hae ja lue sertifikaatti aina, kun näet "X":n.

NEC/CEC-merkinnät (Pohjois-Amerikka)

Pohjoisamerikkalainen merkintä noudattaa eri muotoa:


Class I, Division 1, Groups B, C, D, T4

Tai vyöhykejärjestelmässä:


Class I, Zone 1, AEx db IIC T4 Gb

"A"-etuliite ennen "Ex":ää tarkoittaa, että laite on sertifioitu pohjoisamerikkalaisiin standardeihin (NEC/CEC) vyöhykejärjestelmää käyttäen.

Aiheeseen liittyvät sivut

  • Suojausmenetelmät — kunkin menetelmän yksityiskohtainen kuvaus
  • Kaasuryhmät — mitä IIA/IIB/IIC tarkoittaa
  • Lämpötilaluokat — mitä T1–T6 tarkoittaa
  • EPL — mitä Ga/Gb/Gc tarkoittaa
  • Standardit — mikä standardi koskee kutakin merkintäelementtiä
  • Sertifiointi — miten sertifikaatit ja merkinnät myönnetään
