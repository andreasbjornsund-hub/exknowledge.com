Så läser du Ex-märkningar på utrustning

Varje certifierad Ex-enhet bär en märkning som kodar dess skyddstyp, gas-/dammgrupp, temperaturklass och utrustningsskyddsnivå. Att förstå dessa märkningar är avgörande för val, installation och inspektion av utrustning.

ATEX-märkningsstruktur

ATEX-utrustning bär en strukturerad märkning enligt direktiv 2014/34/EU:


CE 0539  ⟨Ex⟩  II 2G  Ex db IIC T4 Gb
│   │     │     │  │    │  │  │   │  │
│   │     │     │  │    │  │  │   │  └─ EPL (Gb = Hög, zon 1)
│   │     │     │  │    │  │  │   └──── Temperaturklass (T4 = max 135 °C)
│   │     │     │  │    │  │  └──────── Gasgrupp (IIC = väte/acetylen)
│   │     │     │  │    │  └─────────── Skyddstyp (db = flamskyddad, EPL b)
│   │     │     │  │    └────────────── Ex-symbol (explosionsskyddad)
│   │     │     │  └─────────────────── Kategori (2G = Kategori 2, gas)
│   │     │     └────────────────────── Utrustningsgrupp (II = ytindustrier)
│   │     └──────────────────────────── ATEX hexagonal "Ex"-symbol
│   └────────────────────────────────── Anmält organs nummer
└────────────────────────────────────── CE-märkning

IECEx-märkningsstruktur

IECEx-märkningar följer IEC 60079-0:


Ex db IIC T4 Gb
│  │  │   │  │
│  │  │   │  └─ EPL
│  │  │   └──── Temperaturklass
│  │  └──────── Gasgrupp
│  └─────────── Skyddstyp med nivåsuffix
└────────────── Ex-symbol

IECEx-märkningar inkluderar inte CE-märkning, gruppnummer eller kategori. Istället ger IECEx-certifikatnumret spårbarhet:


IECEx BAS 12.0034X
│     │   │
│     │   └─ Certifikatnummer + revision (X = särskilda villkor)
│     └───── ExCB-kod (BAS = Baseefa, UK)
└─────────── IECEx-systemidentifierare

Kombinerade märkningar

Utrustning med flera skyddsmetoder:


Ex db eb IIC T4 Gb

Huvudkropp: flamskyddad (db). Plintfack: ökad säkerhet (eb).


Ex db [ib Ga] IIC T4 Gb

Flamskyddat hölje (Gb) som innehåller egensäkra kretsar (Ga). Parenteserna anger ES-kretsarna inuti.


Ex db IIC T6...T4 Gb

Temperaturklassen varierar med omgivningstemperaturen. T6 vid standardomgivning, T4 vid förhöjd omgivning.

Dubbel gas- och damm-märkning

Utrustning certifierad för både gas och damm:


II 2G  Ex db IIC T4 Gb
II 2D  Ex tb IIIC T125°C Db

Båda raderna visas på typskylten. Utrustningen kan användas i gas zon 1 eller damm zon 21.

Verkliga exempel

Exempel 1: Kopplingsdosa


CE 0539 ⟨Ex⟩ II 2G Ex eb IIC T5 Gb
  • ATEX Grupp II, Kategori 2G (zon 1 gas)
  • Ökad säkerhet (Ex eb)
  • Alla gasgrupper (IIC)
  • T5 (max 100 °C yta)
  • EPL Gb

Exempel 2: LED-strålkastare


CE 0539 ⟨Ex⟩ II 2G Ex db eb IIC T4 Gb
                    II 2D Ex tb IIIC T135°C Db
  • Dubbelklassad: gas (zon 1) och damm (zon 21)
  • Flamskyddad kropp med plintbox med ökad säkerhet
  • Alla gasgrupper (IIC), alla dammgrupper (IIIC)
  • Gas: T4 (max 135 °C), Damm: max 135 °C

Exempel 3: ATEX-certifierat iPhone-fodral (Xshielder)


⟨Ex⟩ II 2G Ex db IIC T4 Gb
     II 2D Ex tb IIIC T135°C Db
  • Zon 1 gas + zon 21 damm
  • Flamskyddat + dammskyddat hölje
  • Täcker alla gasgrupper och dammgrupper
  • IP65-klassat, kompatibelt med trådlös laddning

Exempel 4: Egensäker transmitter


CE 0539 ⟨Ex⟩ II 1G Ex ia IIC T4 Ga
  • Kategori 1G — lämplig för zon 0
  • Egensäker (högsta nivån, Ex ia)
  • Alla gasgrupper (IIC)
  • T4 (max 135 °C)
  • EPL Ga (mycket högt skydd)

Exempel 5: Zon 2-motor


CE 0539 ⟨Ex⟩ II 3G Ex nA IIB T3 Gc
  • Kategori 3G — enbart zon 2
  • Icke-gnistande (Ex nA)
  • Gasgrupper IIA och IIB (inte IIC)
  • T3 (max 200 °C)
  • EPL Gc

Suffixet "X" är viktigt — det betyder att det finns särskilda villkor dokumenterade i certifikatet som måste beaktas vid installation och användning. Hämta alltid och läs certifikatet när du ser "X".

NEC/CEC-märkningar (Nordamerika)

Nordamerikansk märkning följer ett annat format:


Class I, Division 1, Groups B, C, D, T4

Eller i zonsystemet:


Class I, Zone 1, AEx db IIC T4 Gb

Prefixet "A" före "Ex" anger att utrustningen är certifierad enligt nordamerikanska standarder (NEC/CEC) med zonsystemet.
