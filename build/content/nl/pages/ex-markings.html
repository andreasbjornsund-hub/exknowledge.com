Ex-markeringen op apparatuur lezen

Elk gecertificeerd Ex-apparaat draagt een markering die het beschermingstype, de gas-/stofgroep, temperatuurklasse en het apparaatbeschermingsniveau codeert. Het begrijpen van deze markeringen is essentieel voor het selecteren, installeren en inspecteren van apparatuur.

ATEX-markeringsstructuur

ATEX-apparatuur draagt een gestructureerde markering conform Richtlijn 2014/34/EU:


CE 0539  ⟨Ex⟩  II 2G  Ex db IIC T4 Gb
│   │     │     │  │    │  │  │   │  │
│   │     │     │  │    │  │  │   │  └─ EPL (Gb = Hoog, Zone 1)
│   │     │     │  │    │  │  │   └──── Temperatuurklasse (T4 = max. 135°C)
│   │     │     │  │    │  │  └──────── Gasgroep (IIC = waterstof/acetyleen)
│   │     │     │  │    │  └─────────── Beschermingstype (db = drukvast, EPL b)
│   │     │     │  │    └────────────── Ex-symbool (explosiebeveiliging)
│   │     │     │  └─────────────────── Categorie (2G = Categorie 2, Gas)
│   │     │     └────────────────────── Apparaatgroep (II = bovengrondse industrie)
│   │     └──────────────────────────── ATEX-zeshoekig "Ex"-symbool
│   └────────────────────────────────── Nummer aangemelde instantie
└────────────────────────────────────── CE-markering

IECEx-markeringsstructuur

IECEx-markeringen volgen IEC 60079-0:


Ex db IIC T4 Gb
│  │  │   │  │
│  │  │   │  └─ EPL
│  │  │   └──── Temperatuurklasse
│  │  └──────── Gasgroep
│  └─────────── Beschermingstype met niveauachtervoegsel
└────────────── Ex-symbool

IECEx-markeringen bevatten geen CE-markering, groepsnummer of categorie. In plaats daarvan biedt het IECEx-certificaatnummer traceerbaarheid:


IECEx BAS 12.0034X
│     │   │
│     │   └─ Certificaatnummer + revisie (X = bijzondere voorwaarden)
│     └───── ExCB-code (BAS = Baseefa, VK)
└─────────── IECEx-schema-identificatie

Gecombineerde markeringen

Apparatuur met meerdere beschermingsmethoden:


Ex db eb IIC T4 Gb

Hoofdlichaam: drukvast (db). Klemcompartiment: verhoogde veiligheid (eb).


Ex db [ib Ga] IIC T4 Gb

Drukvaste omhulling (Gb) met intrinsiek veilige circuits (Ga). De haakjes geven de IS-circuits binnenin aan.


Ex db IIC T6...T4 Gb

Temperatuurklasse varieert met omgevingstemperatuur. T6 bij standaard omgeving, T4 bij verhoogde omgeving.

Dubbele gas- en stofmarkering

Apparatuur gecertificeerd voor zowel gas als stof:


II 2G  Ex db IIC T4 Gb
II 2D  Ex tb IIIC T125°C Db

Beide regels verschijnen op het typeplaatje. De apparatuur kan worden gebruikt in gas Zone 1 of stof Zone 21.

Praktijkvoorbeelden lezen

Voorbeeld 1: Aansluitdoos


CE 0539 ⟨Ex⟩ II 2G Ex eb IIC T5 Gb
  • ATEX Groep II, Categorie 2G (Zone 1 gas)
  • Verhoogde veiligheid (Ex eb)
  • Alle gasgroepen (IIC)
  • T5 (max. 100°C oppervlak)
  • EPL Gb

Voorbeeld 2: LED-schijnwerper


CE 0539 ⟨Ex⟩ II 2G Ex db eb IIC T4 Gb
                    II 2D Ex tb IIIC T135°C Db
  • Dubbel goedgekeurd: gas (Zone 1) en stof (Zone 21)
  • Drukvaste behuizing met klemkast met verhoogde veiligheid
  • Alle gasgroepen (IIC), alle stofgroepen (IIIC)
  • Gas: T4 (135°C max.), Stof: 135°C max.

Voorbeeld 3: ATEX-gecertificeerde iPhone-hoes (Xshielder)


⟨Ex⟩ II 2G Ex db IIC T4 Gb
     II 2D Ex tb IIIC T135°C Db
  • Zone 1 gas + Zone 21 stof
  • Drukvaste + stofbeschermde omhulling
  • Dekt alle gasgroepen en stofgroepen
  • IP65 geclassificeerd, compatibel met draadloos laden

Voorbeeld 4: Intrinsiek veilige transmitter


CE 0539 ⟨Ex⟩ II 1G Ex ia IIC T4 Ga
  • Categorie 1G — geschikt voor Zone 0
  • Intrinsiek veilig (hoogste niveau, Ex ia)
  • Alle gasgroepen (IIC)
  • T4 (135°C max.)
  • EPL Ga (zeer hoge bescherming)

Voorbeeld 5: Zone 2-motor


CE 0539 ⟨Ex⟩ II 3G Ex nA IIB T3 Gc
  • Categorie 3G — alleen Zone 2
  • Niet-vonkend (Ex nA)
  • Gasgroepen IIA en IIB alleen (niet IIC)
  • T3 (200°C max.)
  • EPL Gc

Het "X"-achtervoegsel is belangrijk — het betekent dat er bijzondere voorwaarden in het certificaat zijn gedocumenteerd die bij installatie en gebruik moeten worden nageleefd. Haal altijd het certificaat op en lees het wanneer u "X" ziet.

NEC/CEC-markeringen (Noord-Amerika)

Noord-Amerikaanse markeringen volgen een ander formaat:


Class I, Division 1, Groups B, C, D, T4

Of in het zonesysteem:


Class I, Zone 1, AEx db IIC T4 Gb

Het "A"-voorvoegsel voor "Ex" geeft aan dat de apparatuur gecertificeerd is volgens Noord-Amerikaanse normen (NEC/CEC) met het zonesysteem.
